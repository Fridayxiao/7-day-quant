# 7-day-quant｜7天速成量化入门

> 面向零基础：用 7 天建立“量化交易”最小可用认知与实践路径。

## 你将学到什么
- 量化交易的基本流程：**想法 → 数据 → 因子/信号 → 回测 → 风控 → 迭代**
- 常见策略类型：趋势、均值回归、动量、配对交易
- 如何避免“看起来赚钱、实盘亏钱”的常见坑
- 如何在 7 天内完成一个可复现的入门级策略练习

## 学习前准备（30 分钟）
- 数学：均值、方差、相关系数（会看懂即可）
- 编程：Python 基础语法
- 工具：
  - Python 3.10+
  - Jupyter Notebook（可选）
  - 常用库：`pandas`、`numpy`、`matplotlib`

---

## 7 天学习计划

### Day 1：建立量化思维
**目标**：理解“量化不是预测未来，而是管理概率”。

- 认识收益率、波动率、最大回撤、夏普比率
- 明确策略评估基准：收益、风险、稳定性
- 输出：写下你对“好策略”的 3 条标准

---

### Day 2：拿到并清洗数据
**目标**：学会处理最常见的行情数据。

- 数据字段：时间、开高低收、成交量
- 数据清洗：缺失值、重复值、复权、时间对齐
- 输出：完成一份可直接用于回测的日线数据

---

### Day 3：实现第一个信号
**目标**：做出“可解释”的交易信号。

- 示例：双均线策略（短期均线上穿长期均线买入）
- 加上最基本交易规则：仓位、手续费、滑点
- 输出：生成每个交易日的买卖信号（0/1 或 -1/0/1）

---

### Day 4：手写简易回测
**目标**：理解回测核心，不依赖黑盒。

- 根据持仓与价格变化计算策略净值
- 统计关键指标：年化收益、最大回撤、夏普比率
- 输出：策略净值曲线 + 指标表

---

### Day 5：加入风险控制
**目标**：从“会交易”到“活下来”。

- 常见风控：止损、仓位上限、单笔风险控制
- 过拟合警惕：参数不要调到“完美贴合历史”
- 输出：加入 1 个风控规则并比较前后表现

---

### Day 6：做一次参数与稳健性检查
**目标**：验证策略不是“碰巧有效”。

- 参数扫描（例如均线窗口）
- 不同时间区间验证（训练/验证分离）
- 输出：记录“哪些参数区间稳定、哪些不稳定”

---

### Day 7：形成你的最小策略项目
**目标**：完成可复现的入门闭环。

- 统一整理：数据处理、信号、回测、风控、结果
- 写结论：策略适用场景、失效场景、下一步优化方向
- 输出：一个可复现的入门策略说明（含图表与结论）

---

## 新手常见误区（务必避开）
1. **只看收益，不看回撤**
2. **忽略手续费和滑点**
3. **未来函数（用了未来数据）**
4. **过拟合（参数过度调优）**
5. **样本内很好，样本外崩溃**

## 推荐进阶方向
- 多因子框架
- 组合优化与资产配置
- 事件驱动策略
- 机器学习在量化中的正确使用方式
