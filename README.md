# 7-day-quant

这是一个面向量化研究入门的 7 天学习文档仓库。内容从金融与 Python 数据分析基础开始，逐步进入组合风险、时间序列信号、因子策略、回测偏差控制，并在第 7 天整合为一个小型研究项目。

## 学习目录

1. [Day 1 - 经济学与金融学基础](day-01-finance-economics-foundations.md)
2. [Day 2 - Python 金融数据分析](day-02-python-financial-data-analysis.md)
3. [Day 3 - 组合与风险](day-03-portfolio-risk.md)
4. [Day 4 - 时间序列与信号](day-04-time-series-and-signals.md)
5. [Day 5 - 因子思维与策略设计](day-05-factor-thinking-and-strategy-design.md)
6. [Day 6 - 回测与偏差控制](day-06-backtesting-and-bias-control.md)
7. [Day 7 - 最终项目与研究报告](day-07-final-project-research-report.md)

## 建议学习方式

- 每天按文档顺序完成概念、练习和复盘。
- 不要跳过风险、交易成本、数据延迟和回测偏差部分。
- 第 7 天的目标不是追求最好看的收益曲线，而是完成一个规则清晰、可复现、能解释局限性的研究流程。

## 适合人群

- 有编程基础，想系统入门量化研究的人。
- 想把金融概念、Python 数据分析和策略回测串起来的人。
- 需要一套短周期量化研究训练材料的人。

## 免责声明

本仓库内容仅用于学习和研究，不构成投资建议。任何策略示例都应在充分理解风险、交易成本、数据质量和现实执行约束后再进一步评估。
