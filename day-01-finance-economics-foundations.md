## 0. 入门者如何读今天的内容

如果你是计算机科学背景，今天最重要的不是背金融术语，而是把金融问题翻译成你熟悉的系统思维。

可以把金融市场看成一个复杂的分布式系统：

- 资产是系统里的对象。
- 价格是对象在某个时点的状态。
- 收益率是状态变化。
- 利率、通胀、政策和盈利预期是外部输入。
- 投资者、交易员、基金、银行和做市商是不同类型的参与者。
- 交易规则、保证金、交易成本和流动性是系统约束。

你今天不需要做到：

- 推导资产定价模型。
- 记住所有宏观指标发布时间。
- 判断明天市场涨跌。
- 读懂所有金融新闻。

你今天需要做到：

- 看到一个金融术语时，能说出它大概影响哪类资产。
- 看到一段价格序列时，知道应该转成收益率再分析。
- 看到一个策略结果时，知道要同时问收益、风险、回撤和成本。
- 知道哪些问题现在不必深入，哪些问题会影响后续回测。

建议学习方法：

1. 先读概念，不停下来查太多外部资料。
2. 每读完一个概念，用一句话写自己的解释。
3. 遇到公式先看直觉，再看符号。
4. 先接受“金融里很多关系是概率性的”，不要期待像算法题一样有确定答案。

## 1. 经济系统的基本参与者

一个简化经济体里有几类核心参与者：

- 家庭：提供劳动、消费商品、储蓄资金。
- 企业：雇佣劳动、生产商品、投资扩张。
- 政府：征税、支出、发行债务、提供公共服务。
- 中央银行：影响基础利率、货币供给和金融条件。
- 金融机构：银行、基金、券商、保险公司、做市商、交易所。
- 投资者和交易者：承担风险，交换资产，形成价格。

金融市场的核心功能不是让人“炒股”，而是：

- 融资：企业和政府从投资者手中获得资金。
- 定价：市场把未来现金流、风险和流动性折成今天的价格。
- 风险转移：不同参与者可以交换自己不想承担的风险。
- 流动性：资产可以更容易地买卖。
- 信息聚合：价格反映大量参与者对未来的判断。

量化研究的切入点是：如果价格是信息、预期、风险偏好和交易机制共同作用的结果，那么价格序列和基本面数据里可能存在可检验的统计结构。

### 一个从现实到价格的例子

假设美国通胀数据高于市场预期。可能发生的链条是：

```text
通胀高于预期
央行可能更久维持高利率
债券收益率上升
债券价格下降
股票估值压力上升
高成长股票可能更敏感
美元可能走强
黄金和新兴市场资产可能受影响
```

这条链条不是必然规律，而是常见传导路径。真实市场还会看：

- 数据高了多少。
- 市场之前是否已经预期到了。
- 央行讲话是否改变。
- 其他经济数据是否同时变弱。
- 投资者当时仓位是否拥挤。

所以金融研究经常不是问“这件事一定导致什么”，而是问：

```text
在类似条件下，这类事件平均会怎样影响不同资产？
这种影响是否足以覆盖交易成本？
```

## 2. 宏观变量如何影响资产

量化研究即使只做股票，也不能完全忽视宏观变量。你至少要理解下面几个变量。

### GDP

GDP 衡量一个经济体在一段时间内生产的最终商品和服务总价值。它反映经济活动强弱。

直觉：

- GDP 增速强，企业收入和利润可能改善。
- 增速过热，可能带来通胀和加息压力。
- 增速变弱，股票盈利预期可能下降，债券可能受益于降息预期。

### 通胀

通胀是物价总体水平上升。常见指标包括 CPI 和 PPI。

直觉：

- 温和通胀通常和增长并存。
- 高通胀会侵蚀货币购买力。
- 高通胀可能迫使央行加息。
- 利率上升会提高贴现率，压低远期现金流的现值。

### 利率

利率是资金的价格。无风险利率通常是资产定价的地基。

直觉：

- 利率上升，债券价格通常下降。
- 利率上升，股票估值的贴现率上升，高成长股往往更敏感。
- 利率下降，融资成本降低，风险资产可能受益。

### 汇率

汇率是两种货币的相对价格。

直觉：

- 本币贬值可能有利出口企业，但会提高进口成本。
- 外币收入较多的公司会受到汇率影响。
- 全球资产配置必须考虑汇率收益和汇率风险。

### 央行政策

央行通过政策利率、公开市场操作、资产购买和沟通影响市场预期。

量化研究中常见思路：

- 利率变化影响债券期限结构。
- 通胀和利率影响股票行业轮动。
- 政策冲击可能改变资产相关性。

### 宏观变量的入门速记表

| 变量 | 它在衡量什么 | 常见市场影响 | 初学者容易误解 |
|---|---|---|---|
| GDP | 经济总产出 | 增长强通常利好盈利，但也可能带来加息压力 | GDP 好不等于股票一定涨 |
| 通胀 | 物价上升速度 | 高通胀可能推高利率，压制估值 | 通胀下降不等于价格下降，只是涨得慢 |
| 利率 | 资金价格 | 利率上升通常压低债券价格和远期现金流估值 | 利率影响所有资产，不只影响债券 |
| 失业率 | 劳动力市场强弱 | 失业上升可能意味着增长变弱，也可能带来降息预期 | 坏经济数据有时会让市场上涨 |
| 汇率 | 货币相对价格 | 影响跨国企业收入、进口成本和全球资金流 | 本币贬值并非对所有公司都坏 |
| 央行政策 | 金融条件松紧 | 改变贴现率、风险偏好和流动性 | 市场看的是预期差，不只是政策本身 |

“预期差”是入门必须理解的概念。市场价格反映的是投资者之前的预期。一个数据本身好或坏并不够，关键是它比市场预期更好还是更差。

## 3. 资产价格的核心直觉

最重要的定价直觉是：

```text
资产价格 = 未来现金流的预期现值
```

如果资产未来能产生现金流，那么今天的价格大致取决于三件事：

1. 未来现金流有多少。
2. 这些现金流有多不确定。
3. 投资者用多高的贴现率把未来折算到今天。

一个简化公式：

```text
Price = Expected Cash Flow / (Discount Rate - Growth Rate)
```

这不是让你机械套公式，而是建立直觉：

- 现金流预期上升，价格倾向上升。
- 贴现率上升，价格倾向下降。
- 长期增长预期上升，价格倾向上升。
- 不确定性上升，投资者要求更高风险补偿，价格倾向下降。

### 贴现率的直观解释

贴现率可以理解成投资者要求的最低回报率。它由几部分组成：

```text
贴现率 = 无风险利率 + 风险补偿 + 流动性补偿 + 不确定性补偿
```

假设一家公司明年预期能给你 100 美元现金流。

如果你要求 5% 回报，那么这笔钱今天大概值：

```text
100 / 1.05 = 95.24
```

如果你要求 10% 回报，那么今天大概值：

```text
100 / 1.10 = 90.91
```

同样的未来现金流，因为要求回报率不同，今天价格不同。这就是为什么利率上升会影响股票、债券、房地产和其他资产。

### 风险溢价是什么

风险溢价是投资者为了承担风险而要求的额外收益。

如果短期国债几乎无风险，年化收益是 4%，而股票预期年化收益是 8%，那么粗略看股票风险溢价是：

```text
8% - 4% = 4%
```

这 4% 不是保证收益，而是投资者愿意承担盈利波动、估值下跌、经济衰退和流动性风险的补偿。

## 4. 主要资产类别

### 股票

股票代表公司所有权的一部分。股票投资者承担公司经营风险，并分享利润增长。

股票收益来源：

- 盈利增长。
- 估值提升。
- 股息。
- 回购。

股票风险：

- 盈利下滑。
- 估值压缩。
- 行业竞争。
- 利率上升。
- 流动性下降。

### 债券

债券是借款凭证。债券投资者借钱给发行人，获得利息和本金偿还。

债券关键变量：

- 票息。
- 到期收益率。
- 久期。
- 信用风险。
- 通胀风险。

重要直觉：

```text
利率上升 -> 旧债券吸引力下降 -> 旧债券价格下降
利率下降 -> 旧债券吸引力上升 -> 旧债券价格上升
```

久期越长，债券价格对利率越敏感。

### ETF

ETF 是交易所交易基金。它像股票一样交易，但通常持有一篮子资产。

常见 ETF：

- SPY：美国大盘股票。
- QQQ：纳斯达克 100。
- TLT：长期美国国债。
- GLD：黄金。

ETF 很适合入门量化研究，因为：

- 数据相对容易获取。
- 分散化程度较高。
- 可以代表不同资产类别。
- 策略解释更清楚。

### 期货

期货是约定未来某时间以某价格买卖标的资产的合约。

常见标的：

- 股指。
- 国债。
- 原油。
- 黄金。
- 农产品。
- 外汇。

期货有杠杆和保证金制度。初学阶段只需要理解概念，不建议一周内深入期货实盘细节。

### 期权

期权是权利，不是义务。

- Call option：买入标的的权利。
- Put option：卖出标的的权利。

期权价值来自：

- 标的价格。
- 行权价。
- 到期时间。
- 波动率。
- 利率。
- 分红。

本周主线是量化研究，不会深入期权定价。你只需要知道期权是非线性风险工具。

### 初学者资产类别对比

| 资产 | 你买到的是什么 | 主要收益来源 | 主要风险 | 入门研究难度 |
|---|---|---|---|---|
| 股票 | 公司所有权 | 盈利增长、估值提升、股息 | 盈利下滑、估值压缩 | 中 |
| 债券 | 借款合同 | 利息、本金偿还、利率下降带来的价格上涨 | 利率上升、违约、通胀 | 中 |
| ETF | 一篮子资产 | 底层资产组合收益 | 底层资产风险、跟踪误差 | 低 |
| 期货 | 标准化远期合约 | 标的价格变化 | 杠杆、保证金、展期 | 高 |
| 期权 | 买入或卖出权利 | 方向、波动率、时间价值变化 | 非线性亏损、定价复杂 | 高 |
| 现金/货币基金 | 流动性资产 | 短期利息 | 通胀侵蚀、再投资风险 | 低 |

本周选择 ETF，是因为它在“足够真实”和“足够简单”之间比较平衡。

## 5. 收益率

价格本身通常不是研究对象，收益率才是。

简单收益率：

```text
r_t = P_t / P_{t-1} - 1
```

如果昨天价格是 100，今天价格是 103：

```text
r = 103 / 100 - 1 = 0.03 = 3%
```

对数收益率：

```text
log_return_t = ln(P_t / P_{t-1})
```

对数收益率的好处是时间可加：

```text
ln(P_2 / P_0) = ln(P_1 / P_0) + ln(P_2 / P_1)
```

实际研究中：

- 做组合和回测时，简单收益率更直观。
- 做统计建模时，对数收益率经常更方便。
- 两者在日频小波动下差异很小。

### 为什么不要直接研究价格

价格有三个问题：

1. 不同资产价格水平不可比。SPY 是 500 美元，GLD 是 200 美元，并不代表 SPY 更贵。
2. 价格通常有趋势，不稳定。很多统计方法会被趋势误导。
3. 投资者真正关心的是资金增长比例，而不是价格点数。

收益率把问题改成：

```text
我的钱增长了百分之多少？
```

这让不同资产、不同时间段和不同策略可以比较。

### 简单收益和累计收益的关系

如果连续三天收益是：

```text
Day 1: +10%
Day 2: -10%
Day 3: +5%
```

累计收益不是：

```text
10% - 10% + 5% = 5%
```

而是：

```text
(1 + 0.10) * (1 - 0.10) * (1 + 0.05) - 1
= 1.1 * 0.9 * 1.05 - 1
= 3.95%
```

这是复利效应。回测里必须用连乘，而不是简单相加。

## 6. 风险的基本语言

### 波动率

波动率是收益率的标准差。它衡量资产价格变化的不稳定程度。

日收益率标准差：

```text
daily_vol = std(daily_returns)
```

年化波动率：

```text
annual_vol = daily_vol * sqrt(252)
```

这里 252 是美股一年大致交易日数量。不同市场可以调整。

### 最大回撤

最大回撤衡量从历史高点跌到低点的最大跌幅。

计算直觉：

```text
wealth_index = cumulative product of (1 + returns)
running_peak = historical maximum of wealth_index
drawdown = wealth_index / running_peak - 1
max_drawdown = minimum drawdown
```

最大回撤很重要，因为投资者真实感受到的痛苦往往不是波动率，而是账户从高点跌了多少。

### 相关性

相关性衡量两个资产收益率一起变动的程度。

- 接近 1：经常同涨同跌。
- 接近 0：线性关系弱。
- 接近 -1：经常一个涨一个跌。

组合管理里，低相关资产可以降低组合整体波动。

### Beta

Beta 衡量资产相对市场的系统性风险。

简化定义：

```text
stock_return = alpha + beta * market_return + error
```

如果 beta = 1.2，市场上涨 1% 时，该资产平均上涨约 1.2%。这只是统计关系，不是确定规律。

### Alpha

Alpha 是扣除市场风险暴露后仍然存在的超额收益。

在量化研究中，很多人说“寻找 alpha”，实际意思是寻找不能简单用市场风险解释的收益来源。

### Sharpe Ratio

Sharpe ratio 衡量单位波动率获得的超额收益。

简化版本：

```text
Sharpe = annualized_return / annualized_volatility
```

更严格版本会减去无风险利率：

```text
Sharpe = (annualized_return - risk_free_rate) / annualized_volatility
```

注意：

- Sharpe 高不代表一定好。
- 它假设波动率能代表风险。
- 它对非正态收益、尾部风险和流动性风险不够敏感。

### 收益指标和风险指标的关系

初学者常犯的错误是只看收益最高的资产或策略。专业研究至少同时看四件事：

| 指标 | 回答的问题 | 如果只看它的风险 |
|---|---|---|
| 年化收益 | 长期平均赚多少 | 可能忽略巨大回撤 |
| 年化波动率 | 过程有多颠簸 | 可能低估尾部风险 |
| 最大回撤 | 最痛苦时亏多少 | 不反映长期收益能力 |
| Sharpe | 每单位波动获得多少收益 | 对非线性和极端风险不敏感 |

一个策略年化收益 20%，最大回撤 70%，未必比年化收益 10%、最大回撤 15% 的策略更好。量化研究不是找最高收益，而是找风险调整后更合理的收益。

### 风险不是一个数字

风险至少包括：

- 市场风险：整个市场下跌。
- 利率风险：利率变化影响资产价格。
- 信用风险：借款人违约。
- 流动性风险：想卖时卖不掉或冲击很大。
- 模型风险：你的模型假设错了。
- 数据风险：数据源或清洗有问题。
- 操作风险：代码、执行、订单或系统出错。

本周主要处理市场风险、波动率、回撤和交易成本。其他风险会在第二周逐步补充。

## 7. 量化研究的基本问题

量化研究不是“我觉得会涨”，而是把市场假设转成可检验的规则。

一个完整研究问题应该长这样：

```text
假设：过去 12 个月表现较强的资产，在未来 1 个月仍有较高收益。
数据：SPY, TLT, GLD, QQQ 的日线价格。
信号：过去 252 个交易日的累计收益率。
组合：每月持有动量最高的 2 个资产。
交易：月末调仓。
成本：每次换仓按成交金额的 0.05% 估算。
评估：年化收益、年化波动率、Sharpe、最大回撤。
验证：和等权买入持有比较。
```

你要避免这样的研究问题：

```text
找一个赚钱策略。
```

这不是研究问题，因为它没有假设、数据、规则、成本和验证方式。

## 8. 有效市场和量化研究的张力

有效市场假说认为，价格已经反映可获得的信息。越成熟、越流动、越被研究的市场，越难获得稳定超额收益。

但市场并非完全有效，可能原因包括：

- 投资者行为偏差。
- 风险补偿。
- 机构约束。
- 流动性限制。
- 信息处理成本。
- 交易成本。
- 市场结构变化。

量化研究的任务不是证明市场永远错，而是问：

```text
某个规则的历史表现能否用风险、成本、数据偏差和运气解释？
```

如果不能完全解释，它才可能是一个值得进一步研究的策略。

## 11. 手算练习

假设某资产 5 天价格如下：

```text
Day 0: 100
Day 1: 102
Day 2: 101
Day 3: 105
Day 4: 103
```

任务：

1. 计算每天简单收益率。
2. 计算 4 天累计收益率。
3. 找出期间最高点和最低点。
4. 计算从最高点到之后最低点的回撤。

参考计算：

```text
Day 1 return = 102 / 100 - 1 = 2.00%
Day 2 return = 101 / 102 - 1 = -0.98%
Day 3 return = 105 / 101 - 1 = 3.96%
Day 4 return = 103 / 105 - 1 = -1.90%

Cumulative return = 103 / 100 - 1 = 3.00%

Running high after Day 3 = 105
Drawdown on Day 4 = 103 / 105 - 1 = -1.90%
```

## 12. Python 小练习

创建一个 notebook，运行：

```python
import numpy as np
import pandas as pd

prices = pd.Series(
    [100, 102, 101, 105, 103],
    index=pd.date_range("2026-01-01", periods=5, freq="D"),
    name="price",
)

returns = prices.pct_change().dropna()
wealth = (1 + returns).cumprod()
running_peak = wealth.cummax()
drawdown = wealth / running_peak - 1

summary = pd.DataFrame(
    {
        "price": prices,
        "return": prices.pct_change(),
        "wealth": (1 + prices.pct_change().fillna(0)).cumprod(),
        "running_peak": (1 + prices.pct_change().fillna(0)).cumprod().cummax(),
    }
)
summary["drawdown"] = summary["wealth"] / summary["running_peak"] - 1

print(summary)
print("Mean daily return:", returns.mean())
print("Daily volatility:", returns.std())
print("Max drawdown:", summary["drawdown"].min())
```

你不需要记住所有代码，但要理解每一列的金融含义。

## 13. 新手常见误区

1. 把价格高低当成贵不贵。价格高低本身没有意义，要看收益率、估值和现金流。
2. 把高收益当成好策略。没有风险和回撤，收益没有可比性。
3. 把相关性当成固定常数。相关性会随市场环境变化。
4. 把回测当成未来保证。回测只是历史模拟。
5. 把数学公式当成现实规律。金融公式通常依赖假设。
6. 忽略交易成本。很多微弱信号会被成本吃掉。
7. 看到一个指标就急着优化。先理解经济含义，再做参数测试。
